# Password Generator

A simple Python program that generates a random password
based on user-selected numbers of letters, symbols, and digits.

## How it works

- User chooses how many letters, symbols, and numbers to include
- Password is generated using random selection
- Different generation approaches are demonstrated
- Final password is shown as a string

## Concepts Used

- Python lists
- for loops
- random.choice()
- random.shuffle()
- string concatenation
