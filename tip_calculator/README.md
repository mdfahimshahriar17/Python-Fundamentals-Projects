# Tip Calculator

A simple Python program that calculates how much each person should pay
after adding a tip and splitting the bill.

## How it works

- Takes the total bill amount
- Adds a percentage-based tip
- Splits the total amount among people
- Shows the final amount per person rounded to 2 decimal places

## Mathematical concepts used

- Percentage calculation
- Addition
- Division
- Order of operations (BODMAS)
- Rounding to two decimal places

## Python concepts used

- input()
- type casting (int, float)
- arithmetic operators
- formatted string output
