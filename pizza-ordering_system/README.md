# Python Pizza Delivery

A simple Python program that calculates the final price of a pizza
based on size and selected add-ons.

## How it works

- User selects pizza size (S, M, or L)
- Base price is set according to the size
- Optional add-ons (pepperoni and extra cheese) are added to the price
- The final bill is displayed

## Mathematical concepts used

- Addition
- Conditional pricing

## Python concepts used

- if / elif / else conditions
- input()
- string methods (lower())
- arithmetic operators
- formatted output
